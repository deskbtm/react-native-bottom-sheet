# Changesets

Contributors add a changeset when a PR includes user-facing changes to `@deskbtm/react-native-bottom-sheet`.

```bash
pnpm changeset
```

Follow the prompts to pick a semver bump (`patch` / `minor` / `major`) and write a short summary. Commit the generated file under `.changeset/` with your PR.

Maintainers: see [docs/releasing.md](../docs/releasing.md).
